# Goyal Tyres & Rubbers - Inventory Management App

A native Android inventory management system built with Kotlin and Jetpack Compose for tracking rubber stock and sales.

## Features
- **Dashboard** with real-time statistics
  - Total rolls in stock
  - Total weight (kg)
  - Total revenue from sales
  - Pie chart showing stock distribution by rubber type
  
- **Stock Management**
  - Add rubber stock with name, rolls, weight, and cost
  - View all current stock
  - Drop entire stock table

- **Sales Tracking**
  - Record sales with rubber name, rolls, weight, and amount
  - View complete sales history with timestamps
  - Track total revenue

- **Database Export**
  - Export database to Downloads folder
  - View database location

- **Built with:**
  - Kotlin
  - Jetpack Compose
  - Material Design 3
  - Room Database
  - Navigation Component
  - MVVM Architecture

## Requirements
- Android Studio Hedgehog or later
- Android SDK 34
- Minimum SDK 24 (Android 7.0)

## How to Build
1. Open the project in Android Studio
2. Wait for Gradle sync to complete
3. Click "Run" or press Shift+F10

## How to Run
- Connect an Android device via USB with USB debugging enabled, OR
- Use an Android emulator from Android Studio
- Click the Run button in Android Studio
